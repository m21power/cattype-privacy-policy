# Privacy Policy for CatType

**Effective Date:** June 2026  
**App Name:** CatType  

---

## 1. Introduction

CatType is a mobile typing practice application developed to help users improve typing speed and accuracy. This Privacy Policy explains how the application handles user data.

By using CatType, you agree to the terms described in this policy.

---

## 2. Information We Collect

CatType does **not collect, store, or transmit any personally identifiable information**.

We do not collect:
- Name
- Email address
- Phone number
- Location data
- Contacts
- Device identifiers for tracking

---

## 3. Local Data Storage

CatType stores limited data **locally on your device only**, including:

- Typing performance metrics (WPM, accuracy)
- Session history
- User preferences (e.g., theme settings)

This data:
- Remains stored locally on your device
- Is not transmitted to any server
- Is not accessible by the developer

You may delete this data at any time by uninstalling the application.

---

## 4. Internet Usage

CatType is designed to function fully offline and does not require an internet connection.

The application does not:
- Send data over the internet
- Communicate with external servers
- Perform background network requests

---

## 5. Third-Party Services

CatType does not use any third-party services, including but not limited to:
- Analytics providers
- Advertising networks
- Crash reporting tools
- Tracking SDKs

---

## 6. Data Security

Since all data remains locally on the device, users maintain full control over their information. The application does not store or process data externally.

## 7. Children's Privacy

CatType does not knowingly collect any personal information from users of any age, including children.
